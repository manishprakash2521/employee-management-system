# employee-management-system
 
A Spring Boot based Employee Management System that provides REST APIs for managing employee records.
 
## Tech Stack
- Java
- Spring Boot
- Hibernate / JPA
- MySQL
- Maven
 
## Features
- Add Employee
- Update Employee
- Delete Employee
- Get Employee Details
- REST APIs
 
## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Database Layer
